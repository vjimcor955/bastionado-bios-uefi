La BIOS (o su versión más moderna, UEFI) es el primer software que se ejecuta en un dispositivo cuando se enciende. Su función es gestionar y reconocer el hardware instalado y establecer el proceso de arranque del sistema operativo. Sin embargo, si no se protegen adecuadamente sus configuraciones, puede convertirse en un punto de vulnerabilidad, permitiendo que un usuario no autorizado manipule el arranque del sistema.

Esta guía ofrece pasos prácticos para configurar la seguridad en la BIOS/UEFI, previniendo accesos no deseados y reforzando la integridad del sistema desde el momento en que se inicia.

Para proteger la BIOS/UEFI, es necesario configurar contraseñas de administración y arranque, evitando que cualquier usuario sin autorización acceda a las configuraciones críticas del sistema o inicie el dispositivo.

  • Contraseña de administrador: Configura una contraseña de administrador que proteja el acceso a la configuración de la BIOS/UEFI.
  • Contraseña de arranque: Configura una contraseña para el arranque, esto evitará el inicio del sistema sin autorización.

Negative : Nota: Para servidores, esta opción puede ser menos práctica ya que requiere la contraseña en cada arranque o reinicio.

Establecer un orden de arranque seguro asegura que el sistema intente arrancar únicamente desde unidad de almacenamiento configurada, impidiendo el acceso desde otros dispositivos.

Configura como primera opcion la unidad de almacenamiento donde se ecuentra el sistema operativo y, para una mayor seguridad, desactiva el resto de unidades.

Negative : Deshabilitar arranque desde USB: Evita que se pueda iniciar desde unidades USB externas. Esto bloquea el acceso por medios extraíbles que podrían cargar sistemas externos y vulnerar la seguridad.

Activar el Secure Boot (solo disponible en BIOS/UEFI modernas), bloquea el arranque de software no firmado. Es especialmente útil en sistemas que ejecutan Windows o algunas distribuciones de Linux compatibles.

Accede a la configuración de "Secure Boot" y actívala para limitar el arranque a sistemas verificados y firmados.

Smart USB protection

La proteccion inteligente de USB consiste en bloquear el copiado de datos desde el ordenador a un USB. Tiene varias opciones:

  • Read Only: El usuario puede copiar datos desde la unidad USB a la máquina, pero no desde la maquina al USB
  • No access: El usuario no puede hacer uso de unidades USB en windows.

Contraseña de discos duros

Para una mayor seguridad, se pueden configurar contraseñas para proteger los discos duros desde el menu "Seguridad".
